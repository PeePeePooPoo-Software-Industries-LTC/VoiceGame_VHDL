◆ audio_get_average()

int audio_get_average ( )

Calculates the average audio sample of the samples in the current FIFO buffer

It is not possible to know over how many samples this average was calculated, however it is gauranteed to be within 1 and 128 samples

Note
If the sample size within the FIFO is below 1, a negative integer will be returned
Returns
The average audio sample
